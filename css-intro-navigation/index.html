Create nav links in a website

There are several techniques to navigate in a website. The semantic tag for navigation is the <nav> tag. You can navigate through your website by using relative addressing. For our examples, we will use the unordered list tag (<ul>).

  • Page to page within the same folder
  • Page to page between folders
  • Staying on the current page
  • Going to a specific place on the current page
  • Knock knock it's knuckles

Navigational menus are placed in your <header> semantic area. They may also be placed in your <footer> area as sitemap. The nav tag like other semantic tags should have an <hx> tag. However, this header is optional. If you omit the header tag you will recieve a document outliner error: Untitled area. Until you are shown how to fix this fat L, it will be acceptable.
